# Usage

```
ao-cooldown.exe as|burst|fast|fling|fa attack recharge [burst cycle|fa cycle]
```

Example for Envoy to Chaos:

Burst

```bash
# Look up the burst cycle on auno.org
> ao-cooldown.exe burst 1.2 1.2 4000
16s | 1206
15s | 1230
14s | 1255
13s | 1280
12s | 1305
11s | 1330
10s | 1355
 9s | 1380
```

Fling shot

```bash
> ao-cooldown.exe fling 1.2 1.2

12s | 621
11s | 721

 9s | 921
 8s | 1021
 7s | 1121
```
